## 1. 컴포넌트 기반 개발

React 웹 개발은 컴포넌트(Component) 단위로 구성됩니다.

### ✔ id 와 class 차이
- `id`
  - 특정 요소에만 사용
  - 거의 사용하지 않음 ❌

- `class`
  - 여러 요소에 재사용 가능
  - 권장 방식 ✅

### ✔ 핵심
- `class` 기반 설계가 유지보수에 유리
- 재사용 가능한 컴포넌트 설계 중요
